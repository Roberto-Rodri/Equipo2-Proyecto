# Proyecto: Administrador de Gastos

Este repositorio ha sido creado con fines académicos para la gestión sencilla de finanzas de una tienda.

## Propósito del Proyecto
El objetivo principal de este trabajo es que **3 estudiantes** desarrollen habilidades de:
- **Colaboración** efectiva en equipo.
- Uso y dominio de herramientas de **GitHub** (Control de versiones).

---

## Contenido del Repositorio
En este espacio encontrarás los siguientes recursos:

1. **Administrador de Gastos:** Un sistema sencillo para el control de flujo de caja de una tienda.
2. **Base de Datos:** Un esquema pequeño y optimizado para el almacenamiento de la información.
3. **Archivo de Soporte:** Un documento simple con especificaciones adicionales.

---
*Este proyecto es parte de una práctica de aprendizaje colaborativo.*
